# STALCRAFT JVM Wrapper - Краткая инструкция

## 🎯 Что это?

Обёртка которая **АВТОМАТИЧЕСКИ** оптимизирует Java для STALCRAFT на основе вашего железа.

## ⚡ Быстрый старт (2 минуты)

### Шаг 1: Запусти
```
Открой stalcraft-jvm-wrapper.exe
```

### Шаг 2: Установи IFEO (один раз)
```
Нажми кнопку "INSTALL"
Готово! Теперь ВСЕ запуски игры будут оптимизированы
```

### Шаг 3: Играй
```
Запускай игру как обычно через лаунчер
Обёртка сама перехватит и оптимизирует
```

## 📊 Что делает обёртка

### Определяет твоё железо:
- **RAM** - сколько всего и сколько свободно
- **CPU** - количество ядер
- **Large Pages** - поддержка больших страниц памяти

### Генерирует оптимальные JVM флаги:
- **Heap** 6-16GB (на основе твоей RAM)
- **G1GC** с агрессивными настройками (50ms pause target)
- **GC потоки** = ядра - 2
- **Приоритеты** - HIGH_PRIORITY + boosted I/O

### Перехватывает запуск:
```
Лаунчер → stalcraft.exe → IFEO → Обёртка → Java с оптимизациями
```

## 🔧 Ручной запуск (если нужно)

1. **Выбери папку** - нажми 📁 → `C:\Games\STALCRAFT\game`
2. **Или выбери файл** - нажми 🔲 → `stalcraft.exe`
3. **Или введи вручную**
4. **Нажми "LAUNCH GAME"**

## 📈 Панели информации

### HARDWARE PROFILE
- CPU - количество ядер/потоков
- RAM - полоса использования + цифры
- JVM - рекомендуемый heap size

### GENERATED JVM FLAGS
Показывает ВСЕ флаги которые будут применены:
- Heap (динамический)
- G1GC настройки
- GC потоки
- Metaspace
- Code Cache
- Large Pages (если есть)

### SYSTEM OUTPUT LOG
Все действия с таймстампами:
- 🟢 Зелёный - успех
- 🔴 Красный - ошибка  
- 🔵 Синий - информация

## ⚠️ Важно

### IFEO требует администратора
- Запусти `.exe` от имени администратора
- Или используй "LAUNCH GAME" без IFEO

### Large Pages (опционально)
Для ещё лучшей производительности:
1. `secpol.msc`
2. Local Policies → User Rights Assignment
3. Lock pages in memory → добавь своего пользователя
4. Перезагрузись

Обёртка **САМА** определит и включит Large Pages

## 🐛 Проблемы?

### "Failed to create registry key"
→ Запусти от имени администратора

### "Game launch failed"  
→ Проверь что путь к игре правильный
→ Убедись что java.exe есть в папке

### "Tauri command not available"
→ Запускай через `.exe`, а не через `npm run dev`
