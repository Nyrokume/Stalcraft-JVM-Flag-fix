# STALCRAFT JVM Wrapper - Руководство по использованию

## 📋 Что реализовано

### ✅ Полный функционал из Go версии:

1. **Определение системы**
   - Определение RAM (всего/свободно)
   - Определение количества ядер CPU
   - Определение поддержки Large Pages

2. **Генерация JVM флагов**
   - Динамический расчёт heap (6-16 GB на основе RAM)
   - Оптимизация GC потоков
   - Настройка G1GC параметров
   - Настройка Metaspace и Code Cache
   - Включение Large Pages если поддерживается

3. **IFEO (Image File Execution Options)**
   - Установка IFEO хука (требует администратора)
   - Удаление IFEO хука
   - Проверка статуса IFEO
   - Прозрачный перехват запуска игры

4. **Запуск игры**
   - Выбор папки с игрой
   - Выбор executable файла
   - Фильтрация оригинальных JVM флагов
   - Инъекция оптимизированных флагов
   - Повышение приоритета процесса
   - Повышение I/O приоритета

## 🎮 Как использовать

### Способ A: IFEO (Рекомендуется - один раз настроить и забыть)

1. **Запусти приложение**
2. **Нажми "INSTALL"** - зарегистрирует IFEO хук в реестре Windows
   - Требует прав администратора
   - После этого ВСЕ запуски stalcraft.exe/stalcraftw.exe будут перехвачены
3. **Запускай игру как обычно** через лаунчер
   - Обёртка автоматически перехватит запуск
   - Применит оптимизированные JVM параметры
   - Повысит приоритет процесса

### Способ B: Прямой запуск (Для тестирования)

1. **Выбери папку с игрой**
   - Нажми кнопку 📁 рядом с "GAME DIRECTORY"
   - Выбери папку где установлена игра (например: `C:\Games\STALCRAFT\game`)
   
2. **Или выбери executable**
   - Нажми кнопку 🔲 рядом с "TARGET EXECUTABLE"
   - Выбери stalcraft.exe или stalcraftw.exe
   
3. **Или введи путь вручную**
   - В папку: `C:\Games\STALCRAFT\game`
   - В файл: `stalcraft.exe`

4. **Нажми "LAUNCH GAME"**
   - Игра запустится с оптимизированными JVM параметрами

## 📊 Что делает IFEO

```
Обычный запуск:
Лаунчер → stalcraft.exe → Java с стандартными параметрами

С IFEO:
Лаунчер → IFEO перехват → Обёртка → Java с ОПТИМИЗИРОВАННЫМИ параметрами
```

## ⚙️ Автоматические оптимизации

### Heap Memory
- **≤8 GB RAM**: Стандартные параметры игры
- **>8 GB RAM**: 50% свободной RAM (мин 6GB, макс 16GB)

### GC Потоки
- **ParallelGCThreads**: Ядра CPU - 2 (мин 2)
- **ConcGCThreads**: Parallel / 4 (мин 1)

### G1GC Параметры
- MaxGCPauseMillis: 50ms (агрессивная цель)
- G1HeapRegionSize: 4/8/16/32 MB (на основе heap)
- InitiatingHeapOccupancyPercent: 35%
- Adaptive IHOP: включён

### Приоритеты
- HIGH_PRIORITY_CLASS (0x80)
- Memory Priority: 5 (Normal)
- I/O Priority: 3 (High)

## 🔍 Панель JVM Flags

Показывает все сгенерированные флаги с категориями:
- **Heap** - Размер кучи
- **GC** - Мусорщик
- **G1** - Настройки G1GC
- **Threads** - Потоки GC
- **Metaspace** - Метапространство
- **Code Cache** - Кэш кода
- **Large Pages** - Большие страницы (если доступно)

## 🛠️ Кнопки управления

| Кнопка | Действие |
|--------|----------|
| 🔄 (Hardware) | Обновить информацию о системе |
| 🔄 (Flags) | Перегенерировать JVM флаги |
| INSTALL | Установить IFEO хук |
| UNINSTALL | Удалить IFEO хук |
| VERIFY | Проверить статус IFEO |
| LAUNCH GAME | Запустить игру |
| 📁 | Выбрать папку |
| 🔲 | Выбрать файл |

## 📝 Логи

Все действия записываются в SYSTEM OUTPUT LOG:
- Успешные операции (зелёный)
- Ошибки (красный)
- Информация (синий)

## ⚠️ Важные заметки

1. **IFEO требует прав администратора** - Запусти приложение как администратор
2. **Large Pages** - Включаются автоматически если система поддерживает
3. **Оба способа работают одновременно** - Можно использовать IFEO и прямой запуск
4. **Безопасность** - IFEO можно удалить в любой момент кнопкой UNINSTALL

## 🎯 Рекомендуемый сценарий

1. Запусти приложение от имени администратора
2. Нажми INSTALL (настроить IFEO)
4. Запускай игру как обычно - всё будет оптимизировано автоматически!


## 🐛 Решение проблем

### "Failed to create registry key"
- Запусти приложение от имени администратора

### "Game launch failed"
- Проверь что путь к игре правильный
- Убедись что java.exe существует в папке игры

### "File dialog error"
- Введи путь вручную в текстовое поле
