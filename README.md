# STALCRAFT JVM Wrapper - Tauri Edition

Modern GUI JVM optimization wrapper for STALCRAFT built with Tauri.

## Quick Start

Перейдите в папку [stalcraft-jvm-wrapper/](stalcraft-jvm-wrapper/) и следуйте инструктам.

```bash
cd stalcraft-jvm-wrapper
npm install
npm run dev    # Development
npm run build  # Production
```

## Features

- 🎨 **Modern GUI** - Beautiful dark-themed interface
- ⚡ **Dynamic JVM Flags** - Auto-generates optimal settings based on your hardware
- 🚀 **Process Boosting** - Enhanced CPU, memory, and I/O priority
- 🔧 **IFEO Management** - One-click install/uninstall/status checking
- 🎮 **Direct Launch** - Launch game with optimizations without IFEO
